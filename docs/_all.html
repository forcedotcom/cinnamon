global with sharing class VisualforceByIdLocator extends ElementLocator
DescriptionVisualforceByIdLocator is used to locate an HTML element generated by a Visualforce tag. This locator should be used when you want to locate an element that is not supported by VisualforceLocator.

The following code snippet shows how to retrieve a form WebElement generated by <apex:form> tag with 'storyWorkForm' set to Id attribute. Because VisualforceLocator currently does not support <apex:form> tag, VisualforceByIdLocator is used instead and 'form' HTML tag is passed in as the first argument.

VisualforceLocator

global with sharing class VisualforceLocator extends ElementLocator
DescriptionVisualforceLocator is used to locate an HTML element generated by a Visualforce tag. It must be passed in as an argument to getElement method in PageObject class to find and return a WebElement object that represents a certain HTML element.

The following Visualforce tags are currently supported by VisualforceLocator:

  • apex:selectList
  • apex:outputField
  • apex:pageBlockTable
  • apex:commandButton
  • apex:pageMessages
  • apex:inputField
For other Visualforce tags not listed above, VisualforceByIdLocator must be used to locate an element, instead.
SeeWebElement, VisualforceByIdLocator
